Introducción

Rampa AltitudeCDN ™ OmniCache El producto es un sistema de distribución de video empresarial que utiliza tecnología de caché proxy para reducir el tráfico de video a través de la puerta de enlace de Internet o enlaces de ancho de banda limitado dentro de la red empresarial. Con OmniCache, las computadoras de escritorio cliente y los dispositivos móviles que utilizan Brightcove Player extraerán la transmisión directamente del servidor OmniCache y luego el servidor OmniCache extraerá el flujo de Brightcove CDN, si aún no tiene una copia local en el caché. OmniCache es una solución completamente local que se entrega como un dispositivo de software y es administrada por la consola de administración AltitudeCDN ™ Altimeter.  Omnicache se puede utilizar de forma independiente o en conjunto con AltitudeCDN ™ Multicast + para redes mixtas de unidifusión / multidifusión.

La integración con Brightcove utiliza el Complemento OmniCache Video.js para reescribir la URL de la transmisión en el servidor OmniCache.  Para obtener detalles completos sobre la configuración del servidor OmniCache para su uso con Video Cloud, consulte este Nota técnica de la rampa.

Antes de continuar con las instrucciones a continuación, complete el proceso de inicio rápido de OmniCache con el equipo de Ramp para asegurarse de que OmniCache servidor y la red local está configurada correctamente.

Configuración del jugador

Comience configurando un reproductor Brightcove para usar el complemento OmniCache. Crea un nuevo jugador con la máscara y las opciones deseadas.

Luego, agregue el complemento OmniCache Video.js.

Copia y pega desde aquí:

    https://livetools.ramp.com/omnicache/plugin/brightcove-omnicache-v1.4.js

Finalmente, configure las opciones del complemento OmniCache. Reemplace el valor de proxy "omnicachedemo.ramp.com" con el nombre DNS de su servidor OmniCache. Alternativamente, también puede especificar un equilibrador de carga de DNS para distribuir el tráfico de video entre múltiples servidores OmniCache.

Copia y pega desde aquí:

    {
        "proxy": "omnicachedemo.ramp.com",
        "verbose": true,
        "timeout": 2000
    }

Insertar Brightcove Player

Finalmente, incruste el reproductor Brightcove en su página web utilizando el proceso de publicación estándar del reproductor.
