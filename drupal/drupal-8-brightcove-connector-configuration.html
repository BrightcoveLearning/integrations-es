Conector Drupal 8-Brightcove: Configuración

Introducción

Drupal Brightcove Video Connector le permite administrar videos y reproductores Brightcove Video Cloud dentro de Drupal, e incrustar videos fácilmente en páginas de Drupal. Tenga en cuenta que esto es para Drupal 8. Para Drupal 7, consulte Instrucciones de Drupal 7.

La instalación de Drupal es un proceso complicado que muy fácilmente sale mal. Asegúrese de que su contenido tenga una copia de seguridad antes de continuar.

Conéctese a la cuenta de Brightcove

  1. Para conectar su cuenta de Brightcove, vaya a la página de la API de Brightcove haciendo clic en Configuracióny, a continuación, desplácese hacia abajo hasta Cuenta API de Brightcove opción bajo Medios de comunicación. Esto lo llevará a la página del cliente de la API de Brightcove. Para agregar un nuevo cliente API de Brightcove, haga clic en Agregar cliente API de Brightcove.
  2. La etiqueta puede ser cualquier cosa, pero usar el mismo nombre que el nombre de la cuenta en Brightcove hará que sea más fácil de identificar.
  3. Para obtener la identificación del cliente y el secreto del cliente, deberá registrar una aplicación. En Brightcove Studio, navegue hasta Autenticación API y haga clic en Agregar nueva aplicación.
  4. Complete el nombre y la descripción breve.
  5. Seleccione una cuenta. A continuación se muestran todos los permisos necesarios.
  6. Presiona guardar.
  7. Aparecen el ID de cliente y el secreto del cliente. Asegúrese de copiar y guardar el secreto del cliente en una ubicación segura. No podrás volver a acceder a él.
  8. Para encontrar el ID de la cuenta, vaya a la sección Información de la cuenta de las páginas de administración:
  9. Complete todos los ID y claves y seleccione Jugador predeterminado.
  10. Hacer clic Ahorrar. El cliente API ahora debería aparecer en la lista.

Sincronización

Inicialmente, cuando navega a la Nuevo contenido página y haga clic en Vídeos de Brightcove, no habrá ninguno, aunque la opción para agregar un video estará allí. Eso es porque primero, necesitas sincronizar los datos. Después de la sincronización inicial, la cronología de Drupal mantiene todo actualizado. Para la sincronización manual, haga clic en el Informes en la parte superior, luego haga clic en Descripción general del estado de Brightcove.

Sobre el Descripción general del estado de Brightcove página, golpe Sincronizar todo.

Vaya a la página de videos de Brightcove nuevamente y sus videos deberían estar disponibles.

Configurar permisos

Ahora puede ver y publicar videos, pero los permisos deben establecerse si los usuarios que no son administradores deben interactuar con los videos. Para configurar estos permisos, primero navegue a la página de permisos haciendo clic en Gente y luego Permisos.

Desplácese hacia abajo para encontrar los permisos de Brightcove. Habilite y deshabilite según sus necesidades. Los permisos de Brightcove se configuran automáticamente como solo de administrador, por lo que si desea que otros vean videos, debe otorgar esos permisos.

Ahora puedes pasar a gestión de contenido o incrustación de contenido.
