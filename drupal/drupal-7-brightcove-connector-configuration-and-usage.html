Conector Drupal 7-Brightcove: Configuración y uso

Introducción

Drupal Brightcove Video Connector le permite administrar videos y reproductores Brightcove Video Cloud dentro de Drupal, e incrustar videos fácilmente en páginas de Drupal. Esta página detalla cómo configurar el módulo Brightcove dentro de Drupal 7. Para Drupal 8, consulte Conector Drupal-Brightcove: Configuración.

La instalación de Drupal es un proceso complicado que muy fácilmente sale mal. Asegúrese de que su contenido tenga una copia de seguridad antes de continuar.

Conéctese a la cuenta de Brightcove

  1. Para conectar su cuenta de Brightcove, vaya a la página de la API de Brightcove haciendo clic en Configuracióny, a continuación, desplácese hacia abajo hasta Configuración de Brightcove opción bajo Medios de comunicación. Esto lo llevará a la página de configuración de Brightcove. Para agregar un nuevo cliente API de Brightcove, haga clic en el Clientes API pestaña, luego seleccione Agregar cliente API. Esto lo llevará al formulario para agregar un cliente API.
  2. La etiqueta puede ser cualquier cosa, pero usar el mismo nombre que el nombre de la cuenta en Brightcove hará que sea más fácil de identificar.
  3. Para encontrar el ID de cuenta, busque en la esquina superior izquierda de su Brightcove Studio tablero.
  4. Para obtener la identificación del cliente y el secreto del cliente, deberá registrar una aplicación. En Brightcove Studio, navegue hasta Autenticación API y haga clic en Agregar nueva aplicación.
  5. Complete el nombre y la descripción breve.
  6. Seleccione una cuenta. A continuación se muestran todos los permisos necesarios.
  7. Presiona guardar.
  8. Aparecen el ID de cliente y el secreto del cliente. Asegúrese de copiar y guardar el secreto del cliente en una ubicación segura. No podrás volver a acceder a él.
  9. Complete todos los ID y claves. Presiona guardar.
  10. Si tiene problemas, asegúrese de tener configurada la fecha y la hora e inténtelo de nuevo.

Agregar un campo de video Brightcove

Para agregar videos de Brightcove a sus publicaciones de Drupal, primero agregue un campo de video a la nueva página de contenido. Para hacer esto, vaya a "Estructura" y haga clic en "Tipos de contenido".

Seleccione "Administrar campos" del tipo de contenido al que desea agregar Brightcove Videos. Este proceso se puede repetir para cada tipo de contenido.

Haga clic en "Agregar un nuevo campo". Etiquételo y seleccione "Brightcove" para el tipo de campo y "Videos (cargar / examinar)" para el widget. Presiona guardar.

Su campo de video debería aparecer ahora.

Publicar y administrar videos

A diferencia de Drupal 8, el conector Drupal 7 no tiene un sistema de administración de contenido separado, por lo que la mayor parte de la administración de contenido debe realizarse en el sitio de Brightcove Studio. Sin embargo, desde la página de contenido nuevo, puede ver todas las imágenes presionando navegar, así como cargar otras nuevas.

Hay dos formas de adjuntar videos existentes a una publicación. Si conoce el título del video, escribirlo en el campo Brightcove y permitir que el autocompletado lo encuentre funcionará. También puede presionar el botón de exploración para ver y seleccionar todos los videos.

Al hacer clic en el botón de carga, se abrirá el formulario de carga, lo que le permitirá agregar nuevos videos directamente desde Drupal. Publica la publicación y tu video debería aparecer.
