Conector Drupal 7-Brightcove: Instalación

Introducción

El conector Drupal-Brightcove le permite administrar videos y reproductores de Brightcove Video Cloud dentro de Drupal, e incrustar videos fácilmente en páginas de Drupal. Este artículo explicará específicamente cómo instalar el complemento de Drupal 7. Para Drupal 8, consulte la guía de instalación de Drupal 8.

La instalación de Drupal es un proceso complicado que muy fácilmente sale mal. Asegúrese de que su contenido tenga una copia de seguridad antes de continuar.

Pasos de instalación

Para obtener los mejores resultados, asegúrese de haber instalado Composer. Encuentra instrucciones sobre cómo aquí.

  1. Descargar e instalar el PHP-API-Wrapper. Mover a sites/all/libraries/PHP-API-Wrapper
  2. Navegar a sites/all/libraries/PHP-API-Wrapper en tu consola. Desde dentro de la carpeta, ejecute composer install.
  3. Ahora, descargue el Módulo Brightcove. También necesitará cargar algunos otros módulos para las dependencias:
    • Herramientas del caos
    • API de entidad
    • API de bibliotecas
    • Fecha
    • Entidad de archivo
    • Medios de comunicación
    • Puntos de vista
  4. Mueva todos los módulos al modules carpeta.
  5. Vaya a su sitio de drupal. Navega a la página de Módulos.
  6. Habilite los siguientes módulos: ctools, API de entidad, bibliotecas, fecha, API de fecha, ventana emergente de fecha, medios, fuentes de medios de Internet, entidad de archivo y vistas. Todos estos deberían estar disponibles gracias al paso 3.
  7. ¡Asegúrese de configurar la fecha y la hora! Su módulo Brightcove no funcionará si no lo hace.

Una vez que haya terminado, puede pasar a Configuración.
