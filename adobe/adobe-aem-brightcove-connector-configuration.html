    Conector Adobe-AEM-BrightCove: Configuración

    En este tema se describe la configuración del conector de Brightcove para Adobe Experience Manager (AEM).

    Introducción

    El conector Adobe-AEM-BrightCove le permite administrar vídeos y reproductores de Brightcove Video Cloud dentro de AEM, así como incrustar vídeos fácilmente en páginas AEM.

    Tenga en cuenta que esta guía utiliza AEM 6.3. El conector también funciona con AEM 6.2andAEM 6.4, y los pasos son similares.  

    Pasos de configuración

    Las siguientes secciones le guiarán a través de los pasos para configurar el conector Adobe-AEM-BrightCove mediante la interfaz clásica. Tenga en cuenta que antes de completar estos pasos, el conector debe estar instalado; consulte Adobe-AEM-BrightCove Connector: Instalación

    Desactivar la transcodificación DAM

    De forma predeterminada, al cargar activos de vídeo a la DAM de AEM, se ejecutará ffmpeg en segundo plano para transcodificar el vídeo en una variedad de formatos.  Si no necesita los activos transcodificados en la DAM, debe desactivar la transcodificación para ahorrar tiempo de procesamiento y almacenamiento:

    1. Abra las herramientas de AEM:
    2. Seleccionar flujo de trabajo:
    3. Seleccionar modelos:
    4. Seleccionar vista de lista:
    5. Seleccionar activo de actualización de DAM:
    6. Seleccione Editar:
    7. Haga doble clic en Transcodificación FFmpeg en el flujo de trabajo para editarlo:
    8. Seleccione la ficha Proceso en el cuadro de diálogo Propiedades del paso:
    9. Cambie el proceso a Ninguna operación:
    10. Cambie Aceptar para cerrar el cuadro de diálogo y guardar los cambios

    Si necesita los activos de vídeo transcodificados localmente, deberá asegurarse de que ffmpeg esté instalado en su servidor AEM:

    1. Compruebe que tiene FFmpeg instalado abriendo una línea de comandos y escribiendo: 
      ffmpeg
    2. Si no se encuentra el ffmpeg comando, debe instalarlo:
      • Instalar en Mac
      • Instalar en Windows
      • Instalar en Linux

    Introduzca la información de la cuenta

    Ahora está listo para configurar el conector AEM-BrightCove con información de cuenta de Video Cloud.

    1. Abra el administrador de configuración de AEM Web Console (http://localhost:4502/system/console/configMgr)
    2. Busque en Brightcove Service para localizar el servicio:
    3. Haga clic en el + signo para agregar una nueva configuración:
    4. En el cuadro de diálogo, introduzca valores como se muestra a continuación. Consulte Administración de credenciales de autenticación de API para obtener instrucciones sobre cómo obtener un identificador de cliente y un secreto de cliente. Los permisos que necesita aquí son:
      • A continuación se indican los valores sugeridos para el formulario de configuración:
        • Alias de cuenta: introduzca un nombre para la cuenta que se mostrará en el conector (obligatorio)
        • ID de cuenta: introduzca su ID de editor de Brightcove (obligatorio)
        • ID de cliente: introduzca su ID de cliente de Brightcove desde la página Autenticación de API de Brightcove (obligatorio)
        • Secreto del cliente: ingrese la API Secret del cliente de Brightcove (obligatorio)
        • Grupos permitidos: especifique el grupo que tendrá acceso al Conector (obligatorio) asegúrese de que es un grupo en el que está incluido.
        • Ruta de la tienda de jugadores: mantenga el valor predeterminado o utilice la carpeta de su elección para los jugadores de esta cuenta de Brightcove (obligatorio)
        • ID de reproductor de vídeo predeterminado: mantenga el valor predeterminado o elija otro ID de reproductor (obligatorio)
        • Clave predeterminada del reproductor de vídeo: obsoleta, deje en blanco o utilice el valor del conector anterior
        • ID de jugador de lista de reproducción predeterminada: introduzca un ID de reproductor habilitado para las listas de reproducción si planea utilizar listas de reproducción (obligatorio)
        • Clave de reproductor de lista de reproducción predeterminada: obsoleta, dejar en blanco o utilizar el valor del conector anterior
        • Servidor proxy: deje en blanco o ingrese proxy si utilizaba anteriormente un proxy
        • Ruta de integración de Dam: mantenga el valor predeterminado o utilice la carpeta de su elección para los activos de vídeo de esta cuenta de Brightcove (obligatorio)
        • Perfil de ingesta: especifique el perfil de ingesta de Brightcove que se utilizará para los vídeos cargados; el valor en blanco utilizará el perfil predeterminado. Si no funciona, solicite a Brightcove que se asegure de que se haya definido un perfil predeterminado en el back-end de Video Cloud
    5. Pruebe su configuración.  Vaya a http://localhost:4502/brightcove/admin y seleccione la cuenta de Brightcove que acaba de agregar en el menú desplegable de la esquina superior derecha.  Si no se muestran vídeos, es probable que se deba a una de las siguientes razones:
      1. Es posible que suID de cuenta, ID de cliente y secretode cliente se hayan introducido incorrectamente en la configuración del conector
      2. Sus credenciales de API no se configuraron correctamente en Brightcove
      3. El usuario que ha iniciado sesión actualmente no es miembro de uno de los grupos permitidos en la configuración del conector
      4. Su configuración de red está bloqueando el acceso de la API a los servidores en la nube de Brightcove. Consulte «Debugging Network and API Connection Problemas» para obtener más información.
    6. Si va a utilizar varias cuentas de Video Cloud, repita los pasos anteriores para cada cuenta.

    Configurar jugadores de Brightcove

    La última tarea de configuración consiste en configurar los reproductores de Brightcove.

    1. Vaya a las herramientas de administración del sitio (http://localhost:4502/siteadmin#/content en la interfaz de usuario clásica o http://localhost:4502/sites.html/content en la interfaz táctil)
    2. En la carpeta Sitios web de nivel superior, cree una nueva carpeta que coincida con el Ruta de la tienda de jugadores en la configuración.  El valor predeterminado es un nombre de carpeta en la configuración se llama brightcovetools con una carpeta dentro llamada reproductores.  Si se conecta a varias cuentas de Brightcove, es posible que desee crear una carpeta de jugadores única para cada cuenta.
    3. Dentro de la carpeta de reproductores , cree una nueva página de Brightcove Player:
    4. Después de crear la página, haga clic con el botón secundario en ella y seleccione Propiedades en el menú de la interfaz de usuario clásica; en la interfaz de usuario táctil, seleccione la página y, a continuación, haga clic en Propiedades.
    5. Vaya a la pestaña Brightcove Basic del cuadro de diálogo y a la cuenta de Brightcove y, a continuación, seleccione un jugador de la lista:
    6. Si también desea que varios reproductores estén disponibles para su publicación, repita los pasos anteriores para crear una página para cada reproductor.
    7. Si también desea utilizar listas de reproducción, repita los pasos anteriores para crear una página para un Reproductor de listas de reproducción.
    8. Si utiliza varias cuentas de Brightcove, deberá crear páginas de jugador para cada una.

    Activar esquema de metadatos para la ubicación de activos de Brightcove

    Si utiliza Touch UI con el conector v5.4 o posterior, deberá activar el esquema de metadatos para la ubicación de activos de Brightcove.   Esto no es necesario si solo está usando la interfaz de usuario clásica.  Active el esquema de metadatos siguiendo los pasos siguientes:

    1. Abra las herramientas de AEM:
    2. Seleccionar activos:
    3. Seleccionar esquemas de metadatos:
    4. Seleccione brightcove y haga clic en Aplicar a carpetas:
    5. Si la carpeta brightcove_assets no existe, cree una carpeta denominada brightcove_assets:
    6. Introduzca el nombre «brightcove_assets»:
    7. Seleccione brightcove_assets y haga clic en Aplicar:

    Ya ha completado la configuración del conector AEM-BrightCove. Continúe con el uso del conector.
