Conector Adobe-AEM-Brightcove: Instalación

Este tema cubre la instalación del conector Brightcove para Adobe Experience Manager (AEM).

Introducción

El conector de Adobe-AEM-Brightcove le permite administrar videos y reproductores de Brightcove Video Cloud dentro de AEM, e incrustar videos fácilmente en páginas de AEM.

Tenga en cuenta que esta guía utiliza AEM 6.3. El conector también funciona con AEM 6.2, 6.4 y 6.5 y los pasos son similares.

Empezando

Si aún no tiene una instancia de desarrollo de AEM, puede crear una fácilmente mediante el archivo.jar de inicio rápido de AEM:

java  -Xmx1400M -jar AEM_6.3_Quickstart.jar

Si utiliza este inicio rápido, su instancia de AEM se alojará en: http: // localhost: 4502 de forma predeterminada. Las siguientes secciones asumirán esta ubicación predeterminada, por lo que si está utilizando una ubicación o puerto diferente, deberá ajustar las URL que se muestran en los pasos o utilizar el menú de navegación de AEM en su lugar.

Instalación del conector

Los siguientes pasos lo guiarán a través de la instalación del conector utilizando la interfaz clásica.

  • Vaya a https://github.com/brightcove/Adobe-AEM-Brightcove-Connector/releases/ y descargue el conector de su versión de AEM (el archivo zip brightcove-all...).
  • Inicie su instancia de desarrollo de AEM.
  • Después de iniciar sesión, vaya al Administrador de paquetes CRX (http: // localhost: 4502 / crx / packmgr / index.jsp)
  • Haga clic en el botón Cargar:
  • En el cuadro de diálogo Cargar, busque el archivo zip del conector y haga clic en Aceptar para cargarlo.
  • Cuando el paquete aparezca en la lista, haga clic en Instalar para instalarlo:
  • Vaya a la página de paquetes de la consola (http: // localhost: 4502 / system / console / bundles) para verificar que el paquete esté activo:

Cuando haya completado estos pasos, el conector se instaló correctamente y debe continuar con el Configuración.
