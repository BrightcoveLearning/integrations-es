Instalación de Brightcove Video Connect para SharePoint

Antes de comenzar la instalación, revise la siguiente información sobre los requisitos previos y los permisos:

  • Antes de que un usuario pueda agregar una aplicación para SharePoint, un miembro del grupo de administradores de la granja debe configurar el entorno para admitir aplicaciones para SharePoint. Para más información, ver Configurar un entorno para aplicaciones para SharePoint 2013 o SharePoint 2016 o SharePoint 2019.
  • Un usuario debe tener el Administrar sitio web y Crear subsitios permisos para agregar una aplicación para SharePoint. De forma predeterminada, estos permisos están disponibles solo para los usuarios que tienen el nivel de permiso de Control total o que están en el grupo del propietario del sitio.
  • Cuando un usuario agrega una aplicación para SharePoint, la aplicación solicita los permisos que necesita para funcionar (por ejemplo, acceso a la búsqueda o para crear una lista). A los usuarios que no tienen esos permisos se les informa que no tienen permisos suficientes y no se puede agregar la aplicación. El usuario puede ponerse en contacto con un administrador del sitio o de la granja para ver si el administrador puede agregar la aplicación.
  • Un usuario que inició sesión en un sitio como la cuenta del sistema no puede instalar una aplicación. La cuenta del sistema no puede importar licencias de aplicaciones porque eso podría provocar problemas de rendimiento.

Adición de la aplicación Brightcove Video Connect al catálogo

Una vez que esté configurado el Catálogo de aplicaciones, siga los pasos que se enumeran a continuación:

  1. Verifique que la cuenta de usuario que está realizando este procedimiento sea miembro del grupo de Propietarios o Diseñadores del sitio para el Catálogo de aplicaciones.
  2. En el sitio del catálogo de aplicaciones, haga clic en el Aplicaciones para SharePoint lista.
  3. Sobre el Aplicaciones para SharePoint página, haga clic en nuevo artículo.
  4. En el Escoge un archivo cuadro, haga clic en Navegar y luego busque la carpeta que contiene el BrightcoveConnectorApp expediente.
  5. Seleccione la aplicación y luego haga clic en Abierto.
  6. Hacer clic OK para cargar la aplicación.
  7. En el cuadro Detalles del artículo, proporcione el nombre, el título, la descripción breve, la URL del icono y otras configuraciones para la aplicación. Asegúrese de que el Activado La casilla de verificación está seleccionada para que los usuarios puedan ver la aplicación en sus sitios. Puede seleccionar la casilla de verificación Destacados para enumerar la aplicación en el Presentado vista de contenido del catálogo de aplicaciones.
  8. Hacer clic Ahorrar.

Instalación de la aplicación Brightcove Video Connect en un sitio

Siga los pasos que se enumeran a continuación para instalar el conector Brightcove en un sitio:

  1. Verifique que la cuenta de usuario que realiza este procedimiento sea miembro del grupo Propietarios del sitio.
  2. La aplicación se puede agregar a un sitio de dos maneras:
    1. En cualquier página, desde el Ajustes selección de menú Agregar una aplicación opción.
    2. De Contenido del sitio utilizando el Agregar una aplicación opción.
  3. A continuación, haga clic en "De su organización" y luego haga clic en el BrightcoveConnector Aplicación
  4. En el cuadro de diálogo Conceder permiso, haga clic en "Confiar".
  5. En este punto, el BrightcoveConnector La aplicación para SharePoint se agrega y aparece en la sección Aplicaciones de la lista de contenido del sitio. Ahora está disponible para su uso en el Sitio.

Instalación del servidor proxy para la comunicación de la API de Brightcove

Dado que Connector es una aplicación de JavaScript del lado del cliente, se requiere un servidor proxy para gestionar la comunicación autenticada con la API de Brightcove. Puede encontrar más información en el Documento de la API de Brightcove.

Se proporciona una aplicación proxy .net en el paquete Brightcove Video Connect. El servidor proxy debe implementarse en una ubicación accesible desde la web, como en Microsoft Azure. Por razones de seguridad, el proxy está diseñado para permitir solo conexiones desde sitios web incluidos en el archivo web.config que se instala con el proxy. La clave brightcove: origins en el archivo web.config debe incluir la URL del sitio web de SharePoint para el BrightcoveConnector La aplicación, así como la URL del sitio web del sitio en el que BrightcovePlayer el componente se ejecutará. Vea el ejemplo a continuación:

<add key="brightcove:origins" value="https://brightcovecms-a3da847a1c9e33.sharepoint.com, https://brightcovecms.sharepoint.com" />

Se pueden enumerar varios sitios como una cadena separada por comas.

Lanzamiento de Brightcove Video Connect para SharePoint

Para iniciar la aplicación, el usuario puede hacer clic en el BrightcoveConnector Aplicación dentro de la lista Contenido del sitio, o haga clic en un enlace de menú si así lo configuró el administrador de SharePoint.

En cualquier caso, el usuario será llevado a la página de inicio de la aplicación Connector. El cliente podría configurar esta página para incluir mensajes adicionales específicos de la empresa, enlaces adicionales, etc. Como se describe en la sección de instalación, dependiendo de la pertenencia al grupo del usuario, es posible que vea o no las pestañas Administración de cuentas y Agregar videos.

La navegación principal en la columna izquierda del Conector siempre está presente y permite al usuario navegar por todas las secciones a las que tiene acceso (consulte la sección anterior sobre permisos). A continuación se muestra una breve descripción del propósito de cada componente del conector:

  • Casa - La página de destino cuando ingresa por primera vez al Conector.
  • Administración de cuentas - Proporciona la interfaz para ingresar la información de la (s) cuenta (s) de Brightcove Video Cloud en SharePoint.
  • Agregar videos - Cargue nuevos videos en Video Cloud. También se crea un objeto de video en SharePoint (a excepción del archivo de video real).
  • Administrar videos - Explore y edite videos existentes, así como importe videos de Video Cloud a SharePoint.
  • Administrar listas de reproducción - Cree nuevas listas de reproducción y explore y administre listas de reproducción existentes.

Tambien hay Agregar video desde Brightcove y Editar video componentes que se mostrarán en la navegación principal al realizar esas operaciones.

Vinculación a cuentas de Video Cloud

La sección Administración de cuentas de Connector se utiliza para ingresar la información de cada una de las cuentas vinculadas de Brightcove Video Cloud a las que se accederá en el Sitio. Las cuentas de Brightcove no se pueden crear desde SharePoint, primero deben crearse en Video Cloud. La información de la cuenta de Brightcove debe ingresarse en SharePoint, ya que se utiliza al acceder a la API de Brightcove para comunicarse con Video Cloud. También proporciona los medios para asignar permisos de acceso específicos a una cuenta. Para los clientes empresariales de múltiples cuentas, proporciona control sobre qué usuarios pueden acceder a cada cuenta y si pueden simplemente navegar o cargar videos.

Página de inicio de administración de cuentas

La página de inicio de administración de cuentas muestra la lista actual de cuentas configuradas, junto con los grupos asignados para autores y lectores.

  1. Titulo de cuenta - Cuentas configuradas actualmente (o al menos creadas).
  2. AutoresGrupo - El grupo de SharePoint asignó derechos de "Autor" a la cuenta.
  3. ViewersGroup - El grupo de SharePoint asignó derechos de "Visor" para la cuenta.
  4. Agregar nueva cuenta - Abra la pantalla de creación de cuenta para configurar una nueva cuenta.

Crear / editar una cuenta

Ya sea creando una nueva cuenta o editando una cuenta existente, el proceso y las opciones son los mismos. Todas las propiedades de la cuenta (excepto los grupos Autor y Visores) deben obtenerse primero de la cuenta de Video Cloud. Las siguientes son las propiedades requeridas para cada cuenta.

  1. Nombre de la cuenta - Nombre de la cuenta. Este nombre solo se utiliza en SharePoint, por lo que no es necesario que sea el mismo que en Video Cloud.
  2. DefaultVideoPlayerId - Establece el reproductor predeterminado para videos.
  3. DefaultPlaylistPlayerId - Establece el reproductor de listas de reproducción predeterminado.
  4. ID de la cuenta - El ID de cuenta de Brightcove de Video Cloud.
  5. Identificación del cliente - El ID de cliente de autenticación de API de Brightcove.
  6. Secreto del cliente - El secreto del cliente de autenticación de API de Brightcove.
  7. Ubicación de almacenamiento temporal - La ubicación de almacenamiento temporal que se utilizará durante la carga de videos y pistas de texto en Video Cloud. El menú desplegable permite la selección de Amazon Web Services (AWS) o Dropbox.
  8. ID de clave de acceso de AWS - Si usa AWS, el ID de la clave de acceso se ingresa aquí.
  9. Clave de acceso secreta de AWS - Si usa AWS, la clave de acceso secreta se ingresa aquí.
  10. Nombre del depósito de AWS - Si usa AWS, el nombre del depósito de AWS se ingresa aquí.
  11. ID de cuenta de Dropbox - Si usa Dropbox, el ID de la cuenta de Dropbox se ingresa aquí.
  12. Token de acceso a Dropbox - Si usa Dropbox, el token de acceso a Dropbox se ingresa aquí.
  13. Campos Personalizados - Se utiliza para asignar los campos personalizados creados en Video Cloud para esta cuenta. Mira la sección Asignación de campos personalizados para más información.
  14. Grupo de autores, Grupo de espectadores - Grupos de definición de permisos del lado de SharePoint para la cuenta.
    1. Los miembros del grupo de autores (en este caso el grupo «Propietarios» de SharePoint) tienen la capacidad de realizar funciones en la cuenta que requieren un token de escritura, como agregar un vídeo, editar y actualizar un vídeo existente, crear/editar una lista de reproducción, etc. También pueden realizar todas las funciones de tipo lectura, como navegar y buscar videos y listas de reproducción.
    2. Los miembros del grupo de espectadores (en este caso, el grupo de "visitantes" de SharePoint) solo están autorizados para funciones de tipo lectura, como navegar y buscar videos y listas de reproducción.

Al configurar una cuenta en el conector de Brightcove, los campos personalizados que se han creado en Video Cloud se pueden agregar al perfil de la cuenta en el conector para que aparezcan en las áreas Agregar videos y Administrar videos, además de tener sus valores almacenados. en SharePoint para búsqueda. Cuando se asignan campos personalizados, se enumerarán en la sección Campos personalizados de la pantalla Administración de cuentas, junto con si son campos obligatorios o no:

Para agregar, eliminar o editar campos personalizados, seleccione el botón "Editar campos personalizados". Se muestra la siguiente ventana modal que enumera TODOS de los campos personalizados disponibles para la cuenta. NOTA: esta lista SÓLO estará disponible una vez que la nueva cuenta se guarde por primera vez.

  1. Campo personalizado de Video Cloud - Nombres de TODOS los campos personalizados definidos en Video Cloud para la cuenta seleccionada.
  2. Tipo - El tipo de campo personalizado (Texto o Lista).
  3. Requerido - Indica si el campo personalizado es obligatorio. Esta definición se establece en Video Cloud y no se puede cambiar desde el conector.
  4. Incluir - Haga que el campo personalizado forme parte de la cuenta en el conector para que se muestre al agregar, editar o administrar videos.

Vinculación a Dropbox o AWS S3 para almacenamiento de carga temporal

La API de Brightcove requiere que los archivos (archivos de video, archivos de pistas de texto) que procese Video Cloud se carguen en una ubicación intermedia accesible tanto por el conector de Brightcove como por Video Cloud. AWS S3 y Dropbox son dos servicios recomendados que están disponibles en el conector de Brightcove.

Puede encontrar orientación sobre cómo configurar estos servicios para generar la información requerida en la sección Configuración de la cuenta del conector de Brightcove utilizando los siguientes enlaces:

  • Descripción general: API de ingesta dinámica
    Uso de Dynamic Ingest con S3
    http://docs.aws.amazon.com/AmazonS3/latest/gsg/CreatingABucket.html
    https://www.dropbox.com/developers

NOTA: los archivos cargados en AWS S3 o Dropbox no se eliminan automáticamente. Se recomienda configurar políticas en estos servicios para eliminar archivos automáticamente después de un período de tiempo.

Ventana emergente de ubicación de almacenamiento temporal

Para establecer la ubicación de almacenamiento temporal que se debe utilizar para cargar archivos, seleccione el botón "Ubicación de almacenamiento temporal" y seleccione una de las opciones de almacenamiento del menú desplegable.

  1. Valor - Seleccione Amazon Web Services o Dropbox para la ubicación de almacenamiento temporal.
  2. OK - Registre el valor en la lista de propiedades y cierre el diálogo.

Según la ubicación de almacenamiento que elija, seleccione los botones "AWS Access Key ID", "AWS Secret Access Key" y "AWS Bucket Name" para ingresar los datos de configuración necesarios para AWS S3 o seleccione "Dropbox Account Id" y "Dropbox". Acceder a los botones "Token" para ingresar los datos de configuración necesarios para Dropbox.

Agregar, importar y eliminar videos

El conector de SharePoint permite que un autor de contenido cargue videos desde su máquina local y edite metadatos para los videos.

Agregar videos

La sección Agregar videos del conector se utiliza para cargar nuevos videos en Video Cloud. Todos los datos que se cargan en Video Cloud, con la excepción del archivo de video en sí, también se almacenan en SharePoint, lo que lo hace disponible para las funciones del conector, así como para los resultados de búsqueda de SharePoint. También proporciona al administrador de SharePoint y a los usuarios acceso nativo a los datos sin tener que iniciar sesión en Video Cloud.

Agregar videos admite tanto un solo video como varios videos (carga por lotes). Cuando un usuario accede a Agregar videos (si tiene permiso para hacerlo), se le presentan los siguientes campos que le permiten ingresar los metadatos que se enviarán a Video Cloud:

NOTA: si se carga un video con un estado Inactivo, Pendiente o Eliminado, SharePoint ya no podrá acceder a la información del video para futuras ediciones. Agregar pistas de texto

Se pueden agregar pistas de texto a cualquier video en la primera carga o en las actualizaciones posteriores. Un video puede tener una o más pistas de texto asociadas. El usuario puede seleccionar el tipo de pista de texto que se agregará al video (Carga de archivo, Carga de URL o URL remota). Si se selecciona "Ninguno", la sección no mostrará ninguno de los campos.

IMPORTANTE: Un solo video NO admitirá una combinación de cargas de archivos, cargas de URL o URL remotas. Solo se puede asociar un tipo de pista de texto con cada video.

Subir archivo

Cuando se selecciona "Carga de archivo" para Archivos de seguimiento de texto, se muestran los siguientes campos:

  1. Seleccione el tipo de pista de texto - Permite la selección de carga de archivos, carga de URL o URL remota.
  2. Selector de archivos - Selección del archivo de pista de texto.
  3. Propiedades de pista de texto - Establecer propiedades asociadas con la pista de texto. Al hacer clic en "Agregar", se agrega la pista de texto a los datos del video en la próxima carga / actualización.
    1. Idioma - El código de dos letras (etiqueta de idioma BCP 47 válida) para el idioma de la pista de texto, por ejemplo, "en" para inglés.
    2. Etiqueta - La etiqueta de la pista que será visible para el usuario, como en un menú que enumera los diferentes idiomas disponibles para los subtítulos.
    3. Amable - Uno de los cinco tipos de pistas admitidos enumerados
  4. Lista de pistas de texto agregadas / en cola - Muestra la lista de pistas de texto existentes y nuevas pistas de texto que están en cola para cargar en el próximo guardado. Se puede eliminar una pista de texto haciendo clic en el enlace "Eliminar" y guardando el video.

Carga de URL, carga de URL remota

Al seleccionar una de estas opciones, se reemplaza la carga de "Carga de archivo" con un campo de texto en el que colocar la ruta URL completa al archivo de seguimiento de texto.

  • Subir URL - Ingrese la URL pública donde reside el archivo de pista de texto y luego haga clic en Subir. El archivo se cargará en Video Cloud.
  • URL remota - Ingrese la URL pública donde reside el archivo de pista de texto. La URL debe tener menos de 250 caracteres. Los subtítulos se cargarán desde la URL remota.
  1. Seleccione el tipo de pista de texto - Permite la selección de carga de archivos, carga de URL o URL remota.
  2. Agregar URL - Campo para ingresar la ruta URL completa al archivo de seguimiento de texto.
  3. Propiedades de pista de texto - Permite la configuración de las propiedades asociadas a la pista de texto. Al hacer clic en "Agregar", se agrega la pista de texto a los datos del video en la próxima carga / actualización.
    1. Idioma - El código de dos letras (etiqueta de idioma BCP 47 válida) para el idioma de la pista de texto, por ejemplo, "en" para inglés.
    2. Etiqueta - La etiqueta de la pista que se mostrará al usuario, como en un menú que enumera los diferentes idiomas disponibles para los subtítulos.
    3. Amable - Uno de los cinco tipos de pistas admitidos enumerados.
  4. Lista de pistas de texto agregadas / en cola - Muestra la lista de pistas de texto existentes y nuevas pistas de texto que están en cola para cargar en el próximo guardado. Se puede eliminar una pista de texto haciendo clic en el enlace "Eliminar" y guardando el video.

Administrar videos

La sección Administrar videos del Connector se usa para explorar y administrar videos existentes. Esto incluye la edición y actualización de videos existentes según sea necesario. La función de búsqueda se utiliza para buscar videos por nombre de búsqueda, descripción e información de etiqueta.

Administrar la página de destino de videos

La página de destino Administrar videos muestra la lista de los campos que se pueden usar para explorar o buscar videos en las cuentas disponibles:

  1. Seleccionar cuenta - Esta lista desplegable se completará con todas las cuentas a las que el usuario está autorizado a acceder; esto se habilita a través de la asignación de Grupos de cuentas. El usuario debe seleccionar una cuenta para ver las listas de videos.
  2. Buscar - Filtre los resultados del video por aquellos que contengan el término. Se busca información sobre el nombre, las etiquetas y la descripción junto con los campos de metadatos de SharePoint, si están definidos. Solo es posible buscar en una cuenta a la vez; No se admite la búsqueda cruzada de cuentas.
  3. Mostrar solo SharePoint o solo Video Cloud - Filtre los videos solo para aquellos que están grabados en SharePoint, o aquellos que solo residen en Video Cloud y nunca se han importado sus datos a SharePoint.
  4. Filtros de clasificación / orden - Se utiliza para clasificar y ordenar los resultados según los parámetros enumerados para ayudar a encontrar videos más rápidamente.

Ver una lista de videos en una cuenta

  2. Mostrar solo SharePoint o solo Video Cloud - El propósito de esta opción es filtrar los videos solo a aquellos que están grabados en SharePoint, o aquellos que solo residen en Video Cloud y nunca han tenido sus datos importados a SharePoint.
  3. Filtros de clasificación / orden - Se utiliza para clasificar y ordenar los resultados según los parámetros enumerados para ayudar a encontrar videos más rápidamente.
  4. Controles de paginación - Cada página mostrará hasta 5 listados de videos. Cuando hay más de 5 presentes, aparecerán los controles de paginación.
  5. Listado de videos - Cada lista de videos contiene lo siguiente:
    1. Nombre del video: al hacer clic en el nombre, se abrirá el video para editar sus datos.
    2. Breve descripción
    3. Lista de las etiquetas asociadas con el video (no incluye el ID de referencia aunque esté guardado en Video Cloud como una etiqueta).
    4. El enlace "Editar video". Este enlace (o hacer clic en el Nombre) se usa para abrir el video y editarlo si el usuario actual tiene permiso.
    5. ID de video: Esta es la identificación de video de Video Cloud. Esta identificación aparecerá en la lista una vez que se edite el video (incluso sin cambios) y se guardará por primera vez después de la carga.

Importación de un video solo de Video Cloud a SharePoint

Si hay un video que se cargó directamente en Video Cloud y no a través del Connector, es posible "importar" datos para ese video en SharePoint para que se pueda administrar en SharePoint en el futuro. Para hacer esto, primero busque el video usando la opción de listado "Solo Video Cloud" (1):

Luego haga clic en el nombre del video o en el enlace Editar video para abrir el video en modo de edición como se describe en las siguientes secciones. Una vez que el video se guarda en el modo de edición, los datos también se almacenarán en SharePoint y el video aparecerá en la sección "Solo SharePoint".

IMPORTANTE: el conector de SharePoint solo sincroniza datos de SharePoint a Video Cloud y no de Video Cloud a SharePoint. Esto significa que una vez que importe un video de Video Cloud a SharePoint, cualquier cambio en los metadatos debe realizarse en SharePoint y cualquier cambio en los metadatos realizado en Video Cloud no se actualizará en SharePoint y se sobrescribirá la próxima vez que ese video editado en SharePoint.

La eliminación de videos de SharePoint se realiza fuera del conector. Para eliminar un video en SharePoint, vaya a Contenido del sitio-> Videos donde puede encontrar y eliminar el video.

Si desea importar el video a SharePoint nuevamente en el futuro, las etiquetas que comienzan con "bcsp-" deben eliminarse.

El conector de SharePoint permite a los autores de contenido insertar medios en sus sitios de diversas formas. Este capítulo cubre las opciones disponibles.

Elemento web Brightcove Player para SharePoint local

La solución Sandbox de elementos web de Brightcove Player genera un elemento web que se coloca en una página del sitio en el sitio web interno o externo y reproduce el video de Brightcove. SharePoint en línea restringe la capacidad de los usuarios para agregar elementos web personalizados, por lo que este mecanismo para insertar videos solo está disponible para instalaciones de SharePoint en las instalaciones.

El elemento web también se ha aumentado con algunas de las mismas características en la consola de administración, incluida la selección de cuentas y la búsqueda avanzada con clasificación y orden para encontrar videos y listas de reproducción más rápido.

Para ver las opciones del elemento web del reproductor, la página de SharePoint debe estar en modo "Editar" (1), que mostrará la opción para editar el elemento web (2). Las opciones específicas del elemento web se encuentran en la sección "Configuración de Brightcove" (3) del editor de elementos web.

La expansión de la sección Configuración de Brightcove revela las características actualizadas de la solución de zona de pruebas del elemento web del reproductor.

Seleccionar un video para reproducir en el elemento web

  1. Configuración de Brightcove - Esta sección del editor de elementos web proporciona acceso a los parámetros para configurar el elemento web y seleccionar los videos / listas de reproducción que se reproducirán.
  2. Anchura del jugador, altura del jugador - Establecer las dimensiones del reproductor.
  3. ID de jugador - Especifique el ID del jugador que se utilizará. Es importante que el reproductor seleccionado sea de la misma cuenta que el video seleccionado si se trata de un entorno de múltiples cuentas. No hay un "selector" para el reproductor, por lo que el ID de Video Cloud debe ingresarse manualmente.
  4. ID de video - El campo ID de video se completa con un selector de video. Si el usuario hace clic en el cuadro de puntos suspensivos a la derecha del campo, se muestra la ventana Selector.
  5. Ventana del selector
    • Seleccionar cuenta - Muestra las cuentas a las que el usuario actual tiene permiso para acceder.
    • Videos disponibles - Filtre los resultados del video por aquellos que contengan el término. Se busca información sobre el nombre, las etiquetas y la descripción junto con los campos de metadatos de SharePoint, si están definidos. Solo es posible buscar en una cuenta a la vez; No se admite la búsqueda cruzada de cuentas.
    • Ordenar y ordenar por
    • Ventana de listado de videos - Al hacer clic en un video, se seleccionará para reproducirlo. La lista se limita a mostrar 30 videos, por lo que no todos los videos de la cuenta pueden mostrarse al seleccionar una cuenta por primera vez. El uso de la función de búsqueda permite refinar la lista para encontrar el video deseado.
    • Botón Aceptar - Confirme la selección y complete el campo ID de video con el valor de ID apropiado.

Una vez que se configuran el ID de jugador y el ID de video, el elemento web está configurado y listo para reproducir.

Seleccionar una lista de reproducción para reproducir en el elemento web

El proceso para seleccionar una lista de reproducción para reproducir es casi idéntico al de seleccionar un video.

  1. Configuración de Brightcove - Esta sección del editor de elementos web proporciona acceso a los parámetros para configurar el elemento web y seleccionar los videos / listas de reproducción que se reproducirán.
  2. Anchura del jugador, altura del jugador - Establecer las dimensiones del reproductor.
  3. ID de jugador - Especifique el ID del reproductor de la lista de reproducción que se utilizará. Es importante que el reproductor seleccionado sea de la misma cuenta que el video seleccionado si se trata de un entorno de múltiples cuentas y que está configurado para mostrar una lista de reproducción. No hay un "selector" para el reproductor, por lo que el ID de Video Cloud debe ingresarse manualmente.
  4. ID de lista de reproducción - La identificación de video se completa con un selector de video. Si el usuario hace clic en el cuadro de puntos suspensivos a la derecha del campo, se muestra la ventana Selector.
  5. Ventana del selector
    • Seleccionar cuenta - Muestra las cuentas a las que el usuario actual tiene permiso para acceder.
    • Ordenar y ordenar por
    • Ventana de lista de lista de reproducción - Al hacer clic en una lista de reproducción, se seleccionará para reproducirla. La lista se limita a mostrar 30 listas de reproducción, por lo que no todas las listas de reproducción de la cuenta pueden mostrarse al seleccionar una cuenta por primera vez. El uso de la función de búsqueda permite refinar la lista para encontrar la lista de reproducción deseada.
    • Botón Aceptar - Confirme la selección y complete el campo ID de lista de reproducción con el valor de ID apropiado.

Una vez que se configuran la identificación de la lista de reproducción y la identificación del video, el elemento web está configurado y listo para reproducir.
