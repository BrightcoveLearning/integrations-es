Prepárese para la instalación

Extraiga archivos y carpetas y cópielos a la carpeta de trabajo

  1. Crea una carpeta [Magento_Root]/app/code/Brightcove/Upload
  2. Descargue el archivo de almacenamiento (archivos de código).
  3. Descomprime los archivos / carpetas
  4. Suelta / mueve el contenido de los archivos descomprimidos al directorio en el paso 1

Habilitar extensiones y complementos

Habilite lo siguiente:

  • php bin/magento module:enable Brigtcove_Upload
  • php bin/magento setup:upgrade
  • php bin/magento cache:clean
  • php bin/magento setup:static-content:deploy
  • php bin/magento cache:flush

Configurar las credenciales del módulo Brightcove

Para completar esta parte, deberá obtener las credenciales de cliente para las API que utiliza la extensión. Para obtener detalles sobre cómo crear credenciales, consulte Administrar las credenciales de autenticación de API. Los permisos mínimos que necesitará para las credenciales se muestran a continuación:

  1. Navegar a Tienda -> Configuración -> Brightcove (pestaña izquierda) -> Configuración de vídeo de Brightcove
  2. Ingrese los detalles para la configuración de Brightcove:
    • Identificación del cliente: ID de cliente / usuario REST
    • Secreto del cliente: REST Secreto / contraseña del cliente.
    • URL base de API:
    • IDENTIFICACIÓN: ID de tienda de Brightcove
    • Categoría Video Slot: Para establecer la cantidad de espacios de video en la columna de categoría
    • Tragamonedas de video maestro: Para establecer la cantidad de espacios de video en la columna maestra
    • Ranura de video variante: Para establecer la cantidad de espacios de video en Columna variante
    • Identificación de correo: Para enviar correo electrónico relacionado con el estado de carga masiva
  3. Una vez que todos los detalles se hayan proporcionado correctamente, Ahorrar la página de configuración. Ahora los usuarios deberían poder acceder al módulo personalizado de Brightcove.

Desactivar extensión

Si alguna vez necesita deshabilitar la extensión Brightcove, use los siguientes comandos:

  1. php bin/magento module:disable Brightcove_Upload
  2. php bin/magento setup:upgrade
  3. php bin/magento cache:clean
  4. php bin/magento setup:static-content:deploy
  5. php bin/magento cache:flush
