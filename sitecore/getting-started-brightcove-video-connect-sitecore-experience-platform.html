    Instalación de Brightcove Video Connect para Video Cloud Sitecore CMS

    En este tema se describe el proceso de instalación del módulo de Brightcove Video Connect.

     

    Instalar en pc Video Cloud Sitecore Media Framework 2.1

    Brightcove Video Connect requiere Video Cloud Sitecore Media Framework se instalará primero. los Video Cloud Sitecore Media Framework se instala usando el Video Cloud Sitecore Asistente de instalación. Siga las instrucciones que muestra el asistente de instalación.

    Eventos de página de lanzamiento del flujo de trabajo

    los Video Cloud Sitecore Media Framework incluye una serie de eventos de página que deben liberarse del flujo de trabajo y publicarse antes de que puedan usarse.

    1. Abre el Video Cloud Sitecore Neceser de costura
    2. Seleccione el flujo de trabajo Analytics
    3. Implemente los siguientes eventos de página:
      • Reproducción cambiada
      • Reproducción completada
      • Error de reproducción
      • Reproducción iniciada
    4. Publicar los eventos de página

    Instale Brightcove Video Connect para Video Cloud Sitecore Plataforma de experiencia

    Brightcove Video Connect para Video Cloud Sitecore se instala usando el Video Cloud Sitecore Asistente de instalación. Siga las instrucciones que muestra el asistente de instalación. los Video Cloud Sitecore El complemento está disponible para su descarga a través de un enlace en la tabla anterior. Este archivo zip incluye tanto el código fuente como el paquete de plug-in.  Descomprímalo primero y luego cargue el paquete zip interno en su Video Cloud Sitecore ejemplo.

    Configuración de la base de datos

    Para que el Video Cloud Sitecore Para que los informes de Media Framework funcionen, se debe agregar una nueva tabla de hechos a la base de datos de informes de Experience Database (xDB). Una secuencia de comandos que creará esta nueva tabla está disponible en SDN. Debe ejecutar este script en su base de datos de informes xDB. Este paso debe ser completado por el administrador de la base de datos.

    Nota: Es posible instalar varios Video Cloud Sitecore Conectores de Media Framework. Este script solo debe ejecutarse una vez por base de datos de informes, independientemente de cuántos conectores diferentes haya instalado.

    Configuración de varios servidores

    Video Cloud Sitecore Media Framework se puede instalar en un entorno de creación de contenido de varios servidores. Si tu Video Cloud Sitecore El entorno está configurado de esta manera, debe configurar lo siguiente:

    Servidores de administración de contenido

    Si utiliza el programador para sincronizar medios, sólo un servidor de administración de contenido (CM) debe tener configurado el programador. Quite el trabajo programado de los archivos de configuración de los demás servidores CM. los Video Cloud Sitecore La cola de eventos se encargará de mantener sincronizados los demás servidores CM.

    Nota: La configuración del programador en varios servidores CM no causará ningún problema, pero no es beneficioso hacerlo.

    Servidores de entrega de contenido

    En cada servidor de entrega de contenido (CD), elimine el siguiente índice del archivo de configuración especificado. Este índice solo es necesario en el entorno CM.

    Archivo: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
    Nombre del índice: MediaFramework_Brightcove_Index
     

    Al configurar servidores de CD, puede que le resulte conveniente copiar archivos de configuración de un servidor CM. Si hace esto y ha agregado un programador para sincronizar medios, debe quitar el trabajo programado de los archivos de configuración de los servidores de CD. La sincronización de medios no debe ejecutarse en los servidores de CD. Debería ejecutarse en un servidor CM. La publicación se utiliza para hacer que los elementos multimedia estén disponibles en los servidores de CD.

    Vinculación Video Cloud de cuentas

    Este tema cubre cómo crear una cuenta en Video Cloud Sitecore que se corresponde con un Brightcove Video Cloud cuenta.

     

    Obtener credenciales de cuenta de Brightcove

    Para que Media Framework se conecte a Brightcove Video Cloud , las credenciales de la cuenta de Brightcove deben agregarse a Video Cloud Sitecore . En esta sección se describe cómo localizar las credenciales de la cuenta en Brightcove. La siguiente sección describe cómo agregar esas credenciales a Video Cloud Sitecore .

    1. Inicie sesión en Brightcove Video Cloud . Anote el valor de ID de publicador. Tendrá que ingresar esto en la siguiente sección.Video CloudSitecore
    2. Desplácese a Admin.
    3. Vaya a Autenticación de API.
    4. Registrar una nueva aplicación.
    5. Asigne un nombre a la aplicación, establezca el acceso de cuenta y habilite las API
    6. Anote el ID de cliente y el secreto del cliente. Deberá ingresarlos en Video Cloud Sitecore en la siguiente sección.

    Agregar cuenta de Brightcove a Video Cloud Sitecore

    Esta sección describe cómo agregar una cuenta de Brightcove a Video Cloud Sitecore .

    1. Inicie sesión en Video Cloud Sitecore Escritorio.
    2. Abra la biblioteca multimedia.
    3. Vaya a > Cuentas de Media Library > Media Framework.
    4. Añade un nuevo elemento de la cuenta de Brightcove.
    5. Introduzca los valores ID de publicador, ID de cliente y Secreto de cliente indicados en la sección anterior.
    6. Guarde el artículo.

    Definición de Video Cloud jugadores

    Se necesita un reproductor para mostrar un solo vídeo o una lista de reproducción de vídeos. Un reproductor define qué tipo de capacidades están disponibles para el visitante que quiere ver un video. Video Cloud Sitecore Media Framework permite a los autores de contenido agregar nuevos reproductores, editar reproductores existentes y eliminar reproductores existentes.

     

    Añadir reproductor

    Esta sección describe cómo crear manualmente el Video Cloud Sitecore elemento que corresponde a un jugador.

    1. En Brightcove Video Cloud , vaya a la sección Jugadores.
    2. Seleccione el reproductor en el que desea crear Video Cloud Sitecore . Tenga en cuenta el valor de ID del jugador. Tendrá que ingresar esto en la siguiente sección.Video CloudSitecore
    3. En Video Cloud Sitecore , en la Biblioteca de medios, seleccione el elemento Cuenta de Brightcove.
    4. Seleccione el elemento Cuentas.
    5. Seleccione el elemento Jugadores.
    6. Añada un nuevo elemento del Reproductor de vídeo de Brightcove. Se le pedirá que introduzca el ID del jugador. Introduzca el valor en Brightcove Video Cloud .

    Establecer jugadores predeterminados

    Para obtener una vista previa de videos y listas de reproducción en Video Cloud Sitecore , se deben especificar los reproductores predeterminados. En esta sección se describe cómo configurar los jugadores predeterminados.

    1. En Video Cloud Sitecore , en la Biblioteca de medios, seleccione el elemento Cuenta de Brightcove.
    2. Seleccione el elemento Cuentas.
    3. Seleccione el elemento Configuración.
    4. Establezca los valores para el campo Reproductor de vídeo predeterminado y el campo Reproductor de lista de reproducción predeterminado.
    5. Guardar el elemento

    Editar reproductor

    Los elementos del reproductor se editan como cualquier otro Video Cloud Sitecore artículos.

    Eliminar reproductor

    Los elementos del reproductor se eliminan como cualquier otro Video Cloud Sitecore artículos.

    Nota: Eliminar un jugador de Video Cloud Sitecore no resultará en la eliminación del reproductor de Brightcove Video Cloud .

    Importación, adición, vista previa y eliminación de vídeos

    Video Cloud Sitecore Media Framework permite a un autor de contenido cargar videos desde su máquina local, así como editar y eliminar videos existentes. Se pueden definir nuevas etiquetas para organizar el contenido de vídeo.

     

    Importación de vídeo

    UN Video Cloud Sitecore El autor o administrador del contenido puede activar manualmente el proceso de importación de medios.

    Cuando importas videos, Video Cloud Sitecore reflejará el día actual en que se importaron los videos en lugar de la fecha de carga dentro de Video Cloud . Si está moviendo un Video Cloud Sitecore implementación, puede migrar el contenido anterior y el conector reconocerá estos videos.

    El complemento de Brightcove Video Connect incluye un agente que sincroniza automáticamente los medios de forma regular. El agente se define en la siguiente ubicación:

    Archivo de configuración: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
    Nombre del agente: MediaFramework_Import_Brightcove
    1. Para mostrar la cinta de opciones Marco de medios, haga clic con el botón derecho en el menú superior y seleccione Media Framework.
    2. Haga clic en la cinta de opciones de marco multimedia.
    3. Haga clic en el botón Importar todo.
    4. A medida que se ejecuta el proceso de importación, aparece un cuadro de estado. Cuando desaparece el cuadro de estado, el proceso de importación finaliza.
    5. Para confirmar que se han importado medios, haga clic en la ficha Buscar.
    6. Realice una búsqueda.
    7. Filtrar para vídeos.
    8. Se muestran los resultados de la búsqueda.

    Añadir vídeo

    Video Cloud Sitecore Media Framework permite a los autores de contenido agregar nuevos videos.

    1. Abra la biblioteca multimedia.Video CloudSitecore
    2. Vaya a la cuenta con los vídeos para los que desea crear una lista de reproducción.
    3. Desplácese hasta el elemento Contenido multimedia.
    4. Haga clic en Brightcove Video.
    5. Haga clic en Agregar archivos.
    6. Seleccione el archivo que desea cargar desde su equipo local.
    7. Haga clic en Inicio.
    8. Después de subir el vídeo, cierre la ventana del cargador.

    Definir etiquetas

    Video Cloud Sitecore Media Framework permite a los autores de contenido agregar nuevas etiquetas.

    1. Abra la biblioteca multimedia.Video CloudSitecore
    2. Vaya a la cuenta con los vídeos para los que desea crear una etiqueta.
    3. Desplácese hasta el elemento Etiquetas.
    4. Haga clic en Etiqueta de Brightcove.
    5. Introduzca un nombre de artículo. Esto se convertirá en la etiqueta.

    Editar vídeo y etiquetas

    Los elementos de video y etiquetas se editan como cualquier otro Video Cloud Sitecore artículos.

    Eliminar vídeo y etiquetas

    Los elementos de video y etiqueta se eliminan como cualquier otro Video Cloud Sitecore artículos.

    Buscar vídeos y listas de reproducción

    Los vídeos y listas de reproducción se pueden encontrar buscando cadenas de texto en su nombre y descripciones. Los vídeos también se pueden encontrar buscando cadenas de texto en sus campos personalizados.

     

    Buscando

    Haga clic en la ficha Buscar para buscar medios. Introduzca el texto que desea buscar en los campos de nombre o descripción.

    Haga clic en la lupa para realizar una búsqueda.

    Filtrado

    Los resultados de búsqueda se pueden filtrar por tipo, como vídeo o lista de reproducción, por fecha y por otros atributos.

    Se muestran los resultados de la búsqueda.

    Importación, adición, previsualización y eliminación de listas de reproducción

    Una lista de reproducción es una colección de vídeos en un solo reproductor. Video Cloud Sitecore Media Framework permite a los autores de contenido agregar nuevas listas de reproducción, editar listas de reproducción existentes y eliminar listas de reproducción existentes.

    Cuando importas listas de reproducción, Video Cloud Sitecore reflejará el día actual en que se importaron las listas de reproducción en lugar de la fecha de creación dentro de Video Cloud . Si está moviendo un Video Cloud Sitecore implementación, puede migrar el contenido anterior y el conector reconocerá estos videos.
     

    Añadir lista de reproducción

    Video Cloud Sitecore Media Framework permite a los autores de contenido agregar nuevas listas de reproducción.

    1. Abra la biblioteca multimedia.Video CloudSitecore
    2. Vaya a la cuenta con los vídeos para los que desea crear listas de reproducción.
    3. Desplácese hasta el elemento Contenido multimedia.
    4. Añade un nuevo elemento de la lista de reproducción de Brightcove.
    5. En el campo Nombre, escriba un nombre descriptivo para el vídeo.
    6. En el campo ID de vídeo, seleccione los vídeos que desea incluir en la lista de reproducción.

    Editar lista de reproducción

    Los elementos de la lista de reproducción se editan como cualquier otro Video Cloud Sitecore artículos.

    Eliminar lista de reproducción

    Los elementos de la lista de reproducción se eliminan como cualquier otro Video Cloud Sitecore artículos.

    Agregar video a Video Cloud Sitecore Sitio web

    Video Cloud Sitecore Media Framework permite a los autores de contenido insertar medios en sus páginas de diversas formas. Este capítulo cubre las opciones disponibles.

     

    Editor de texto enriquecido

    Los campos de texto enriquecido permiten a los autores de contenido insertar contenido con formato en sus páginas. Los autores de contenido utilizan el editor de texto enriquecido para crear y editar este contenido con formato. Esta sección cubre las características que Video Cloud Sitecore Media Framework se suma al editor de texto enriquecido.

    Insertar medios

    Los medios se pueden incrustar directamente en un campo de texto enriquecido.

    1. En el editor de texto enriquecido, haga clic en el botón Incrustar medio.
    2. Utilice la pantalla de búsqueda para buscar el contenido multimedia que desea insertar. Haga clic en la fila que representa el medio que desea insertar. Haga clic en el > botón Siguiente.
    3. Dependiendo del tipo de medio que seleccione, verá opciones adicionales. Por ejemplo, si selecciona un vídeo, debe especificar el reproductor que debe utilizarse para mostrar el vídeo, junto con las dimensiones del reproductor. Haga clic en Finish.
    4. El medio está incrustado en el campo de texto enriquecido.

    Incrustar Media Link

    Se puede insertar un vínculo a medios en un campo de texto enriquecido.

    1. En el editor de texto enriquecido, haga clic en el botón Insertar vínculo multimedia.
    2. Utilice la pantalla de búsqueda para buscar el contenido multimedia que desea insertar. Haga clic en la fila que representa el medio que desea insertar. Haga clic en el > botón Siguiente.
    3. Introduzca un valor para Título del vínculo. Este es el texto que aparecerá para el enlace multimedia.
      Además, dependiendo del tipo de medio que seleccione, verá opciones adicionales. Por ejemplo, si selecciona un vídeo, debe especificar el reproductor que debe utilizarse para mostrar el vídeo, junto con las dimensiones del reproductor. Haga clic en Finish.
    4. Un vínculo está incrustado en el campo de texto enriquecido.

    Renderizado

    Los modelizados permiten que el contenido se coloque en una ubicación específica de una página. Video Cloud Sitecore Media Framework incluye una representación que permite a los autores de contenido insertar medios en las páginas.

    Incrustar Media Rendering

    1. Utilice el Editor de páginas para agregar un modelizado a un marcador de posición.
    2. Seleccione el modelizado Incrustar medio.
    3. El modelizado se ha agregado al marcador de posición. Ahora haga clic en el botón Incrustar elemento multimedia.
    4. Utilice la pantalla de búsqueda para buscar el contenido multimedia que desea insertar. Haga clic en la fila que representa el medio que desea insertar. Haga clic en el > botón Siguiente.
    5. Dependiendo del tipo de medio que seleccione, verá opciones adicionales. Por ejemplo, si selecciona un vídeo, debe especificar el reproductor que debe utilizarse para mostrar el vídeo, junto con las dimensiones del reproductor. Haga clic en Finish.
    6. Se ha seleccionado el medio para el modelizado.

    Sincronización entre Video Cloud Sitecore y Video Cloud

    Esta sección cubre cómo mantener sincronizados videos y listas de reproducción entre Video Cloud Sitecore para los siguientes escenarios:

    • Cuando se eliminan vídeos y listas de reproducción de Brightcove
    • Cuando se agregan videos y listas de reproducción Video Cloud Sitecore
    • Cuando se eliminan videos y listas de reproducción de Video Cloud Sitecore

    Importación de Brightcove a Video Cloud Sitecore fue cubierto previamente aquí.

    Exportar vídeo

    Dentro Video Cloud Sitecore , Se accede a los medios de Brightcove mediante Video Cloud Sitecore elementos que se crean durante el proceso de importación de medios. Ya que son Video Cloud Sitecore elementos, es posible que los usuarios cambien esos elementos. El proceso de exportación de medios es responsable de exportar cambios desde Video Cloud Sitecore a Brightcove.

    Exportación automática de medios

    Cuando una Video Cloud Sitecore elemento que corresponde a los medios de Brightcove se crea, guarda o elimina en el Video Cloud Sitecore cliente, esos cambios se enviarán a Brightcove.

    Exportaciones en tiempo real

    Cuando se crea un nuevo elemento multimedia en Video Cloud Sitecore , se realiza una exportación en tiempo real de forma predeterminada. Esto significa que la tarea de exportación se realiza inmediatamente.

    Prevención de la exportación automática

    Es posible que no desee que se realicen cambios en Video Cloud Sitecore para ser exportado a Brightcove. Es posible que desee la conexión de Video Cloud Sitecore a Brightcove para ser de solo lectura, donde los cambios de Brightcove se importan a Video Cloud Sitecore pero donde Video Cloud Sitecore los cambios no se exportan a Brightcove.

    Para evitar cambios realizados en Video Cloud Sitecore de exportarse a Brightcove, borre el valor de la siguiente configuración:

    Archivo de configuración: Video Cloud Sitecore .MediaFramework.config
    Nombre de configuración: Video Cloud Sitecore .MediaFramework.ExportDatabases
     

    Nota: Esta configuración se aplica a Video Cloud Sitecore Media Framework, no solo el complemento Brightcove Video Connect. Cambiar esta configuración se aplicará a todos los conectores y a todas las cuentas.

    Limpieza de datos

    Los procesos de limpieza se utilizan para mantener Video Cloud Sitecore y datos de Brightcove sincronizados.

    Video Cloud Sitecore Limpiar

    Video Cloud Sitecore La limpieza es el proceso que se ejecuta cuando se eliminan los medios en Brightcove y el correspondiente Video Cloud Sitecore los elementos deben eliminarse. Dado que Brightcove no sabe nada sobre Video Cloud Sitecore , Brightcove no puede notificar Video Cloud Sitecore ese medio ha sido eliminado. Por lo tanto, Video Cloud Sitecore debe ser proactivo.

    Limpieza manual

    UN Video Cloud Sitecore El autor o administrador del contenido puede activar manualmente el proceso de limpieza haciendo clic en el siguiente botón en el Editor de contenido:

    Limpieza automática

    El complemento de Brightcove Video Connect incluye un agente que ejecuta automáticamente el proceso de limpieza de forma regular. El agente se define en la siguiente ubicación:

    Archivo de configuración: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
    Nombre del agente: MediaFramework_Cleanup_Brightcove

    Limpieza de Brightcove

    La limpieza de Brightcove es el proceso que se ejecuta cuando se elimina un elemento multimedia en Video Cloud Sitecore y los medios correspondientes en Brightcove deben eliminarse o desactivarse. Dado que Brightcove sabe que se eliminó el elemento multimedia, Video Cloud Sitecore puede notificar a Brightcove que los medios correspondientes en Brightcove deben eliminarse o desactivarse.

    Cuando se elimina un elemento multimedia en el Video Cloud Sitecore cliente, se realiza una entrada en la cola de exportación. El proceso es el mismo que cuando se guarda un elemento multimedia.

    Análisis de eventos de reproducción

    los Video Cloud Sitecore Media Framework permite grabar eventos de reproducción de video para su uso en el Video Cloud Sitecore Sistema de marketing digital (DMS). Esta sección cubre las opciones disponibles.

    Eventos de reproducción

    Los eventos de reproducción se utilizan para grabar eventos de página DMS como resultado de acciones del reproductor de vídeo. Por ejemplo, cuando se inicia la reproducción de vídeo, se puede escribir un evento de página DMS.

    Eventos de reproducción predeterminados

    Video Cloud Sitecore Media Framework viene con los siguientes eventos de reproducción:

    • Reproducción iniciada
    • Reproducción completada
    • Error de reproducción

    Configurar eventos de reproducción para la cuenta

    De forma predeterminada, se graban los eventos de reproducción iniciados y completados de reproducción. Esto está habilitado en el elemento Configuración del elemento Cuenta de Brightcove.

    El elemento Configuración tiene un campo denominado Reglas de eventos de reproducción. Se define una regla que indica a Media Framework que grabe los eventos especificados en vídeos y listas de reproducción.

    Deshabilitar eventos de reproducción para la cuenta

    Todos los eventos de reproducción se pueden desactivar para una cuenta específica.

    1. En el Editor de contenido, vaya a la cuenta bajo > Cuentas de Media Library > Media Framework.
    2. Seleccione el elemento Configuración.
    3. El campo Reglas de eventos de reproducción describe las condiciones que deben cumplirse para que se registren los eventos de reproducción. Por defecto, la condición es «where true», que significa «siempre». Haga clic en el botón Editar regla para cambiar la condición o para quitar la regla existente.

    Especificar eventos de reproducción para medios

    Cada lista de reproducción y elemento de vídeo tiene un campo denominado Eventos. Este campo determina qué eventos de reproducción se controlarán para el elemento especificado.

    Nota: Estos eventos solo se registrarán si el elemento Configuración del elemento de la cuenta de Brightcove tiene su campo Reglas de eventos de reproducción configurado con la acción «agregar eventos de elementos multimedia». Véase la sección 3.1.2.

    Crear nuevos eventos de reproducción

    Puede agregar nuevos eventos de reproducción para capturar información más detallada sobre la reproducción de vídeo.

    1. En el Editor de contenido, vaya a > los módulos del sistema > Media Framework Eventos de > reproducción
    2. Añadir un nuevo elemento mediante la plantilla Evento de reproducción
    3. Para el campo Evento de página, seleccione Reproducción cambiada
    4. Para el campo Parámetro, puede introducir un valor en uno de los siguientes formatos:
      1. Número: después de reproducir el video este número de segundos, se graba el evento de la página. Un ejemplo de un valor válido es 15
      2. Porcentaje: después de reproducir el video esta cantidad, se graba el evento de la página. Un ejemplo de un valor válido es 50%
    5. Guardar el elemento
    6. Publicar el elemento
    7. Ahora puedes añadir el evento de la nueva página a tu lista de reproducción y vídeos

    Nota: Recuerde asignar el nuevo evento de reproducción a su lista de reproducción y vídeos. De lo contrario, el evento no se registrará.

    Informes

    los Video Cloud Sitecore Media Framework incluye varios informes relacionados con la reproducción de video.

    Reproducción iniciada

    Este informe muestra una lista de los vídeos que han desencadenado el evento iniciado por el vídeo.

    Reproducción completada

    Este informe muestra una lista de los vídeos que han desencadenado el evento completado del vídeo.

    Reproducción incompleta

    Este informe muestra una lista de los vídeos que han desencadenado el evento iniciado pero no el evento de vídeo completado.
