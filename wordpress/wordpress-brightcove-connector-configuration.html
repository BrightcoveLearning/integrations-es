Conector de WordPress-Brightcove: Configuración

Este tema cubre la configuración del conector Brightcove para Wordpress.

Introducción

WordPress Brightcove Video Connector le permite administrar videos y reproductores Brightcove Video Cloud dentro de WordPress, e incrustar videos fácilmente en páginas de WordPress.

Conéctese a la cuenta de Brightcove

  1. Vaya a la configuración de Brightcove y haga clic en agregar cuenta de Brightcove
  2. El nombre de la fuente puede ser cualquier cosa, pero usar el mismo nombre que el nombre de la cuenta en Brightcove facilitará la identificación.
  3. Para encontrar la identificación de la cuenta, inicie sesión en Brightcove Studio y busque en la esquina superior izquierda de su panel de control.
  4. Para obtener la identificación del cliente y el secreto del cliente, deberá registrar una aplicación. Navegue hasta Autenticación de API y haga clic en Agregar nueva aplicación.
  5. Complete el nombre y la descripción breve.
  6. Seleccione una cuenta. A continuación se muestran todos los permisos necesarios.
  7. Presiona guardar.
  8. Aparecen el ID de cliente y el secreto del cliente. Asegúrese de copiar y guardar el secreto del cliente en una ubicación segura. No podrás volver a acceder a él.
  9. Haga clic en "Verificar credenciales". Si tiene éxito, sus videos deberían mostrarse.

Establecer el perfil de ingesta

La versión 1.4.1 del conector y anterior codifica el perfil de ingesta de Video Cloud para que sea "videocloud-default-v1".  Si la cuenta de Brightcove solo tiene Dynamic Delivery o CAE habilitado, este perfil de ingesta no existirá y la carga de video fallará.  A continuación, se muestran varias opciones para solucionar este comportamiento en el conector.

 

  1. Cree un nuevo perfil de ingesta en la cuenta de Video Cloud con el nombre de perfil predeterminado utilizado en el conector.  Navegue a Admin-> Ingest Profiles y cree un nuevo perfil o duplique un perfil existente y asígnele el nombre “videocloud-default-v1”.  

 

  1. O cambie el nombre del perfil de ingesta en el código fuente del conector determinando primero qué perfil de ingesta existente en Admin-> Ingesta en la cuenta de Video Cloud que le gustaría usar. Haga clic en Mostrar nombre de API y anótelo.
  1. En el servidor de WordPress, edite wp-content/plugins/brightcove-video-connect/includes/api/class-bc-cms-api.php y cambie el perfil del nombre de perfil predeterminado de 'videocloud-default-v1' al nombre de perfil deseado.  Por ejemplo, cambiar
    public function video_upload( $video_id, $video_url, $profile = 'videocloud-default-v1' ) {
    a
    public function video_upload( $video_id, $video_url, $profile = 'multi-platform-standard-static' ) {
  1. O modifique el conector para utilizar el perfil de ingesta predeterminado en la cuenta de Video Cloud. Para hacer esto, edite wp-content/plugins/brightcove-video-connect/includes/api/class-bc-cms-api.php y comenta la línea:
    $data = array( 'profile' => sanitize_text_field( $profile ) );
    con /* */.

Ahora ha completado la configuración del conector WordPress-Brightcove. Proceder a Usando el conector.
